# Ninja Impact
*Un magnifique jeu de plateformes en 2D, développé par Clément KOENIG et Louis DEFFINIS sur Unity*

### Commandes

**Joueur 1 :**
- Se déplacer à gauche : Q
- De déplacer à droite : D
- Sauter : Z
- Invisibilité : C

**Joueur 2 :**
- Se déplacer à gauche : ←
- De déplacer à droite : →
- Sauter : ↑
- Invisibilité : N
